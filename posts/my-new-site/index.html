Hi everyone, this is my first post on my new site. So welcome! I was able to build this site in less than an hour using Hugo.

Hugo Summary

For those who may not know, Hugo is a static site generator. You can store your site content in markdown files, and then apply either a custom or open source theme to your site. Hugo will then take the content files, and translate them to equivalent html pages. You can then upload the generated files to your favorite static site host.

Github CI Integration

Github gives access to Github pages for public repositories. So using Gitlab CI from a private repository (where your site source code / content is), you can build a pipeline to update a public repo that contains the built hugo html files every time you commit to the private repository! A small pain, but worth it if you’re tech savvy and want free web hosting!
